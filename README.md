# SaaS Landing Template

CloudSync 스타일의 SaaS 랜딩 페이지 템플릿입니다.

## 기능
- Navbar (스크롤 감지, 모바일 햄버거)
- Hero Section (그라데이션 배경, CTA 버튼)
- Features Section (6가지 기능 카드)
- How It Works Section (3단계)
- Testimonials Section (고객 후기)
- Pricing Section (3개 플랜, highlighted Pro)
- FAQ Section (아코디언)
- CTA Section (스크롤 트리거 애니메이션)
- Footer (링크, SNS, 약관)

## 기술 스택
- React 18 + Vite
- Tailwind CSS
- lucide-react

## 시작하기
```bash
npm install
npm run dev
```
